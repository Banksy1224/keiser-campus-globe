# Earth textures

These planetary textures (`earth_atmos_2048.jpg`, `earth_normal_2048.jpg`,
`earth_specular_2048.jpg`, `earth_lights_2048.png`, `earth_clouds_1024.png`)
are the standard three.js example set, derived from **NASA Visible Earth**
imagery, which is in the public domain.

- Day map / normals / specular / night lights / clouds are composited by the
  `Globe` component in `src/pages/campus-map.tsx` into a photoreal Earth.
- Source: https://github.com/mrdoob/three.js/tree/master/examples/textures/planets
- NASA Visible Earth: https://visibleearth.nasa.gov/
