# Brand logo assets

Drop the official Keiser University logo files here and they appear automatically.

| File | Used for |
| --- | --- |
| `wordmark-white.svg` | Header logo (white/reverse, over the dark globe). **Primary.** |
| `wordmark-white.png` | (optional) PNG fallback if you don't have SVG — then set the path in `BrandLogo`. |

Notes:
- The header looks for **`brand/wordmark-white.svg`** first; if it's missing, it
  falls back to the styled text wordmark. So nothing breaks until you add it.
- Use the **reverse / white** version of the wordmark since the header sits on a
  dark background. SVG preferred (crisp at any size); ~transparent background.
- The seal and Seahawks marks can be added later (e.g. on the loading screen or
  campus panels) — send them and I'll wire them in.
